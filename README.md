# 금융 상품 추천 시스템 (FinShibainu 기반)

이 프로젝트는 금융 상품 추천을 위한 AI 기반 시스템으로, RAG(Retrieval-Augmented Generation) 기법을 활용하여 사용자 입력을 처리하고 적절한 금융 상품을 추천합니다.
